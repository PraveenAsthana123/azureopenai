### Key Features

| Feature | Description | Benefit |
|---------|-------------|---------|
| **Intelligent Search** | Hybrid vector + keyword search | Find information even with imprecise queries |
| **Grounded AI Responses** | Answers backed by your knowledge base | Trustworthy responses with source citations |
| **Document Processing** | Automatic OCR, chunking, embeddings | Zero manual document preparation |
| **Multi-LLM Support** | Azure OpenAI, Ollama, Anthropic, OpenAI | Flexibility and vendor independence |
| **Enterprise Security** | Azure AD, private endpoints, RBAC | Meet compliance requirements |
| **Scalable Architecture** | Serverless + VM deployment | Handle enterprise workloads |
